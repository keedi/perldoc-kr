CPAN의 경로를 변경

CPAN의 경로를 변경

$ sudo cpan
cpan> o conf urllist
urllist           
0 [ftp://ftp.daum.net/CPAN/]
Type 'o conf' to view all configuration items

cpan> o conf urllist unshift file:///home/keedi/minicpan
cpan> o conf urllist
urllist           
0 [file:///home/keedi/minicpan]
1 [ftp://ftp.daum.net/CPAN/]
Type 'o conf' to view all configuration items

cpan> o conf commit
cpan> reload index
