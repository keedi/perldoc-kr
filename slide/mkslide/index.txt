어떻게 쓰나요?
index.txt 를 만드세요!
텍스트를 입력, 엔터
또 입력, 엔터, 입력, 엔터, ...
저장 -> 종료
.code
do {
    insert_text();
    insert_newline();
} for @number_of_slides;
save_it();
quit_editor();
.code

코드를 넣고 싶다면?
.code
  코드를 넣고 싶다면 .code로 시작
  .code
  #!/usr/bin/perl
  use strict;
  use warnings;

  print "Hello, mkslide.pl";
  .code
  코드 넣기를 그만둔다면 .code로 시작
.code

이제 필요한건?
Make It!
.code
$ make
[C] index.html ...
bin/mkslide.pl index.txt > index.html
$ _
.code
